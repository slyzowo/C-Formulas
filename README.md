# C-Formulas

## installation

Add into your project in the "libs/" folder and after use `#include "libs/CFormulas.c"`

> [!NOTE]
> You do not need to add it to a folder called `libs/` it is just more helpful too.

# Formulas
Theres a few formulas I have added to this library, and I may add more in the future. Heres just a few I have added so far. Also everything returns as a float.

## Perimeter

A general Perimeter function asking for each side, but if you need specific formulas. We have separate functions for those.
```c
Perimeter(Length1, Width1, Length2, Width2)
```

```c
SquarePerimeter(Length)
```

```c
CirclePerimeter(Length)
```

```c
RectanglePerimeter(Length, Width)
```

```c
TrianglePerimeter(Length1, Width1, Length2)
```
